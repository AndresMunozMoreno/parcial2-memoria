# Parcial 2 - Administración de Memoria

Proyecto para el curso de Sistemas Operativos, Universidad del Valle.
Escuela de Sistemas y Computación.

## Descripción

Este repositorio contiene dos simuladores implementados en Python bajo el
paradigma Orientado a Objetos:

1. **Simulador de Asignación de Memoria**: implementa las estrategias
   First Fit, Best Fit y Worst Fit.
2. **Simulador de Traducción de Direcciones**: traduce direcciones virtuales
   a direcciones físicas mediante paginación de un nivel.

Ambos simuladores se despliegan mediante Docker y cuentan con una interfaz
de menú por consola.
