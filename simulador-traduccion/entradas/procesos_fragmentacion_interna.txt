#   1. Configurar memoria física en 256KB y tamaño de página en 16KB

#   2. Cargar este archivo (opción 2 del menú).

#   P1,17  -> ceil(17/16)=2 paginas -> reservado=32KB -> frag. interna=15KB

#   P2,33  -> ceil(33/16)=3 paginas -> reservado=48KB -> frag. interna=15KB
#             (mismo peor caso, pero con más páginas en uso)
#   P3,64  -> ceil(64/16)=4 paginas -> reservado=64KB -> frag. interna=0KB
#             (mejor caso: múltiplo exacto del tamaño de página, sin
#             fragmentación interna)
